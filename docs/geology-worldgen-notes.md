# 地质生成与世界高度计划

## 当前 0.1.9 已处理

- 普通原版矿石 placed features 已通过 `neoforge:remove_features` 从 Overworld 生物群系移除，并且同步覆盖 19 个原版 `minecraft:ore_*` placed feature 为 `minecraft:count=0`，防止其他加载顺序或兼容层再次引用它们。
- 原版大矿脉不是 placed feature，而是 `OreVeinifier` 根据 `minecraft:ore_veininess` 噪声硬编码生成铜矿、铁矿和粗矿块；本版本覆盖该噪声为 `[0.0]`，避免新生成区块自然出现原版铜矿、铁矿或粗矿块。
- 旧区块里已经生成的原版铜矿/粗铜块不会被资源包式 worldgen 修正自动清除；判断修复效果时应新建世界或探索新生成区块。
- 矿床相对比例改为“煤和低品位铁较常见，普通铁较常见，铜中等偏少，金/青金石/红石/辰砂偏少，金伯利岩和含钻金伯利岩很稀少”。
- 新增围岩/伴生特征：磁铁矿带凝灰质围岩、黄铜矿带花岗质围岩、含金石英带方解石脉旁岩、金伯利岩带凝灰质角砾围岩、烟煤带砂岩沉积围岩、无烟煤带深板岩/变质围岩。
- 新增自然矿床来源：铝土矿风化壳、磷块岩沉积层、蒸发岩盐层、锡石砂矿、稀土碳酸岩矿体。它们先用稳定 placed feature 表达，再通过机器副产物模拟伴生矿。
- 普通石头、圆石、深板岩、深层圆石、黏土、砂、红砂、沙砾、泥土等都有处理路线；按住 Shift 可看到原版材料的近似组成。
- 0.1.7 修正普通石头/圆石定位：普通石头是 MC 泛化本地地层石料，圆石是破碎采石骨料；两者都不是花岗岩，花岗岩仍作为独立花岗质岩石处理。
- 0.1.8 新增默认高度下的真实地质背景 placed features：沉积砂岩层、石灰质方解石层、花岗质/闪长质侵入体、安山岩/凝灰岩火山带、玄武质镁铁质岩脉。
- 0.1.8 按地层重调矿床高度带：烟煤偏中浅沉积盆地，无烟煤偏深部变质带，磁铁矿偏深部镁铁质/火山带，铜金辰砂绿柱石偏花岗质热液带，铝土/锡砂矿偏近地表，稀土/青金石/红石偏深部。
- 0.1.9 细化原版岩石 Shift 材料卡：普通石头显示为本地混合地层石料，圆石显示为破碎采石骨料，花岗岩显示为明确花岗质矿物组合。
- 新增 `data/earth_on_minecraft/earth/strata/` 下的地层规划与实际默认地层元数据，记录默认 384 高度、未来高地层预设方案和智能核心高度倍率联动契约。它们不会默认覆盖旧世界维度。
- 旧测试存档若含有改名前的 `earth_online:*` 矿床方块，会通过 registry alias 映射到新的 `earth_on_minecraft:*`，避免进入旧区块时被替换成默认方块。

## 当前分布逻辑

| 类型 | 游戏分布 | 现实含义 |
| --- | --- | --- |
| 烟煤含煤岩 | 较常见，中浅层 | 沉积盆地煤层，伴随砂岩类围岩，适合煤化工和早期燃料 |
| 无烟煤含煤岩 | 较少，偏深层 | 变质程度更高的高碳煤，伴随深板岩类围岩，偏石墨/高热值路线 |
| 贫磁铁矿 | 常见，深部镁铁质/火山带 | 低品位铁矿，保证开局可找到铁来源 |
| 普通/富磁铁矿 | 普通到稀少，深部镁铁质/火山带 | 可用铁矿床主体，富矿明显更少 |
| 黄铜矿 | 中等偏少，花岗质热液带 | 铜硫化矿，伴随花岗质/热液围岩和黄铁矿副产 |
| 含金石英、辰砂、绿柱石 | 偏少，花岗质热液带 | 小型热液/伟晶岩/断裂矿物体，作为贵金属和宝石入口 |
| 青金石、红石 | 偏少，深部变质基底 | 深部功能矿物体，红石保留 MC 幻想相 |
| 金伯利岩/含钻金伯利岩 | 稀少到极稀少 | 金刚石管状岩体，先用稀疏矿体表达，不做满地宝石 |
| 铝土矿风化壳 | 中等偏少，近地表风化带 | 铝土矿/赤铁矿/高岭土共生，给铝工业更直接自然入口 |
| 磷块岩沉积层 | 偏少，中浅层 | 沉积磷矿，给磷肥、磷酸和石膏副产路线 |
| 蒸发岩盐层 | 偏少，中浅层 | 盐类沉积，给氯碱、盐卤、钾盐和硼酸盐路线 |
| 锡石砂矿 | 稀少，浅层 | 冲积/河床重矿物富集，给锡石、钛白粉和少量独居石 |
| 稀土碳酸岩 | 很稀少，偏中深层 | 稀土碳酸盐/磷酸盐矿体，给氟碳铈矿、独居石和少量铀矿物 |

## 0.1.8 默认地质背景

新增的背景特征不改变世界高度，也不覆盖维度类型；它们只是让默认高度内的岩石背景更像真实地层：

| 背景特征 | 现实含义 | 主要服务矿床 |
| --- | --- | --- |
| `sedimentary_sandstone_bed` | 沉积盆地砂岩层 | 烟煤、锡砂矿、磷块岩、盐层 |
| `limestone_calcite_bed` | 石灰质/碳酸盐层 | 磷块岩、蒸发岩、含金石英方解石脉 |
| `granitic_intrusion` | 花岗质侵入体 | 黄铜矿、含金石英、辰砂、绿柱石、稀土伴生 |
| `diorite_intrusion` | 中性侵入体 | 热液铜金、过渡性围岩 |
| `andesite_volcanic_belt` | 安山质火山带 | 凝灰岩、磁铁矿、热液矿化 |
| `tuff_volcanic_belt` | 凝灰质火山碎屑带 | 磁铁矿围岩、金伯利岩角砾围岩 |
| `basalt_mafic_dike` | 镁铁质岩脉 | 磁铁矿、镁铁质硅酸盐、未来铬铁矿 |

这些比例不是直接照抄现实储量或年产量。现实比例要经过可玩性换算：基础燃料和低品位铁必须能支撑新手流程；铜要支撑机器但不能像煤一样泛滥；贵金属、宝石、辰砂、含钻金伯利岩需要稀少，否则失去探索意义。

## 现实地层规划

当前世界生成仍使用稳定 placed/configured feature 组合；0.1.8 已经从“单个矿床随机点”推进到“分层地质背景 + 矿床类型”的默认高度模型。下一阶段才考虑把主矿、伴生矿和围岩合并成同一个空间结构生成器。

| 地层/环境 | 生成角色 | 典型岩性 | 典型矿床与伴生 |
| --- | --- | --- | --- |
| 表层风化物 | 地表 1-8 格，随生物群系变化 | 土壤、黏土、砂、沙砾、泥巴、红砂 | 铝土矿风化壳、锡石/独居石砂矿、腐殖质和黏土路线 |
| 沉积盆地 | 中浅层大范围背景 | 砂岩、页岩代理、石灰岩/方解石、蒸发岩 | 烟煤、磷块岩、盐层、未来石油/天然气盆地 |
| 花岗质/热液带 | 断裂和侵入岩附近 | 花岗岩、闪长岩、方解石脉、石英脉 | 黄铜矿、含金石英、辰砂、绿柱石、稀土伴生砂 |
| 镁铁质/火山带 | 火山岩和镁铁质岩带 | 玄武岩、安山岩、凝灰岩、黑石代理 | 磁铁矿、铬铁矿、镁铁质硅酸盐、磁铁矿凝灰质围岩 |
| 变质基底 | 深部主背景 | 深板岩、深层圆石、未来片岩/片麻岩代理 | 无烟煤、青金石、红石矿物、稀土碳酸岩 |
| 地幔来源岩筒 | 稀有竖直结构 | 金伯利岩、凝灰质角砾围岩 | 金伯利岩、含钻金伯利岩 |

普通石头与圆石在这个模型里的含义：

- `minecraft:stone` 是默认主世界里的“本地未分类石料”，不能写成花岗岩化学式。后续高地层预设启用后，它应逐步被更具体的岩性背景替代。
- `minecraft:cobblestone` 是玩家挖掘/破碎后的本地采石骨料，机械成因强于地质成因，因此只能给碎屑粉磨路线。
- `minecraft:granite` 才是花岗质侵入岩，应该保留石英、钾长石、斜长石、云母、少量铁氧化物和独居石/锆石类微量矿物设定。

## 世界高度与智能核心联动

默认 MC 26.2 主世界为 `min_y=-64,height=384`。对真实地层来说这很薄，但强行改维度高度会影响旧存档、其他 mod 和世界生成 Codec，因此不能作为默认热更新。

安全路线：

1. 默认世界继续用 384 高度，并把矿床高度保持在合法范围。
2. 新增“高地层世界预设”作为新世界入口，例如保守 `min_y=-128,height=512`，测试型 `min_y=-256,height=768`。
3. 高地层启用时，必须同步重算所有矿床高度带：表层风化物、沉积盆地、热液带、镁铁质/变质基底和岩筒都要迁移。
4. 如果智能核心提供世界高度倍率，Earth on Minecraft 应通过共享配置/世界预设读取，例如 `base_height=384,multiplier=N`，不要直接编译依赖智能核心 Java 类。
5. 真正启用前必须在新世界检查 `latest.log` 中的 `RegistryDataLoader`、`dimension_type`、`worldgen/placed_feature`、`Failed to parse` 等错误。

## 仍需继续推进

- 真实矿产比例不能直接照搬全球储量或产量，否则 MC 生存会变成找不到资源；后续应按“现实相对丰度 + 玩家可玩性 + 兼容性需求”建立一张可调倍率表，并暴露到配置或世界预设。
- 伴生矿当前先用独立 placed feature、围岩 halo 和处理副产物表达。下一步可做专用矿床生成器，让主矿、伴生矿和围岩在同一个空间结构内成型，而不是多个独立随机点。
- 地层厚度确实比现实薄。默认 MC 26.2 主世界是 `min_y=-64,height=384`，直接硬改会影响旧世界和其他 mod；更安全的路线是提供新的世界预设或显式配置，再把矿床高度、岩性层、沉积层、深部变质带同步迁移。
- 后续可继续加入铅锌硫化矿床、沉积铁矿、石油/天然气盆地、盐丘形态、独居石海滨砂矿，以及更真实的“主矿 + 伴生矿 + 围岩 halo”同体矿床生成器。
